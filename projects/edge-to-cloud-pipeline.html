Edge-to-Cloud Detection Pipeline

Containerized pipeline from camera detection events to real-time web dashboard

Overview

Built a containerized edge-to-cloud pipeline that ingests camera detection events via a FastAPI backend and stores them in MariaDB. A web dashboard using Chart.js and AJAX visualizes time-based detection data in real-time. The system demonstrates a full data flow: from edge device events through API ingestion to persistent storage and interactive visualization.

Python FastAPI MariaDB Docker Chart.js AJAX REST APIs

Architecture

📷

Edge Detection

Camera devices generate detection events (objects, timestamps, confidence scores) as structured payloads pushed to the API.

⚡

FastAPI Ingestion

High-performance async REST API receives detection events, validates payloads, and persists them to the database.

🗄️

MariaDB Storage

Relational database stores all detection events with timestamps, enabling time-based queries and aggregations.

📊

Chart.js Dashboard

Web dashboard with AJAX polling visualizes detection trends, time-series data, and aggregated statistics in real-time.

Key Implementation Details

Containerization

The entire stack is containerized with Docker FastAPI backend, MariaDB database, and the web dashboard all run as separate services orchestrated together, making deployment reproducible and portable.

Data Flow

Detection events flow from edge devices → FastAPI REST endpoint → MariaDB → Chart.js dashboard. Each layer is decoupled, allowing independent scaling and testing.

Real-time Visualization

The dashboard uses AJAX-based polling to fetch the latest data from the API, rendering time-based charts that update automatically. Users can see detection patterns, frequency distributions, and historical trends.

What I Learned

This project solidified my understanding of end-to-end data flow: from event ingestion through API design to persistent storage and visualization. Working with FastAPI's async capabilities and Docker containerization became patterns I reuse in professional work.
